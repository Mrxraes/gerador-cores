Este script automatiza a criação de uma identidade visual instantânea e aleatória.


A história começa com o disparo da função cor(), que atua como um artista em busca de inspiração:

--- > O código define três valores aleatórios entre 0 e 255. Essa mistura de
Vermelho, Verde e Azul gera uma cor única e imprevisível.

--- > A cor resultante é aplicada imediatamente ao fundo do elemento #box, 
transformando o cenário.

--- > Para garantir contraste, o script calcula uma versão levemente mais escura
da cor original (subtraindo valores dos canais RGB) e a aplica ao texto do elemento #analogo.
